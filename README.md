# MEAN Demo

### Dependencies:

- NodeJS
- NPM
- MongoDB or MySQL
    - Select the correct routes in 

### How to Download JS Packages

- `npm install` on `\<root>` 
- `npm install` on `\<root>\client`

### How to Run

- Back-end Express
    - Hit `node server.js`

- Front-end Angular
    - Hit `ng build --watch`
